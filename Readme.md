# macOS Portfolio

My new **macOS-inspired portfolio**, 
built with **Next.js, TypeScript, and Zustand**

### Usage :
```bash
git clone https://github.com/Roshan-Here/Roshan-MacOs.git
cd macos-leo
npm install
npm run dev
```
